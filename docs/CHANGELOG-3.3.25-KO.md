# 3.3.25 변경 내역

## 버전

- 표시 버전을 `3.3.25`로 올렸습니다.
- Android 업데이트 설치가 가능하도록 `versionCode 357`로 올렸습니다.

## 펜 두께

- 펜과 도형 획 두께를 현재 화면 확대율 기준이 아니라 페이지 기준 두께로 고정했습니다.
- 같은 두께 값의 획은 확대해도 페이지 위 실제 두께가 유지되며, 화면에서는 페이지와 함께 자연스럽게 커집니다.
- 기존 `screenWidth`가 저장된 획도 렌더링 시 1배율 페이지 폭 기준으로 해석되도록 수정했습니다.

## S Pen 지우개

- S Pen 버튼이 `MotionEvent`가 아니라 스타일러스 `KeyEvent`로 들어오는 기기에서도 버튼을 누르는 동안만 지우개로 작동하도록 수정했습니다.
- 펜이 화면에 이미 닿아 획이 시작된 상태에서 S Pen 버튼을 눌러도 즉시 현재 획을 지우개 세션으로 전환합니다.
- 버튼을 놓으면 이전 도구로 돌아가며, 지우개가 토글처럼 남지 않도록 했습니다.

## 검증

- 웹 회귀 테스트에 페이지 기준 두께 확대 동작과 S Pen KeyEvent 중간 전환 케이스를 추가했습니다.
- 웹 회귀 테스트와 Android `updateRelease` 빌드를 통과했습니다.
