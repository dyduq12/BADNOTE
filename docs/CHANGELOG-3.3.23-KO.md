# 3.3.23 변경 내역

## S Pen 지우개

- 3.3.23 재배포 APK는 `versionCode 355`입니다. 표시 버전은 3.3.23으로 유지합니다.
- 실제 기기에서 `ACTION_BUTTON_RELEASE`가 버튼 bit를 포함한 채 들어와도 S Pen 버튼을 놓은 것으로 처리하도록 수정했습니다.
- S Pen 버튼 지우개 알림을 화면 가운데에서 왼쪽 위로 옮겨 필기 영역을 덜 가리게 했습니다.
- 버튼을 놓거나 입력이 취소되면 지우개 알림이 자동으로 사라지도록 정리했습니다.
- 펜이 화면에 닿아 이미 획을 그리고 있는 중에도 Android pointerId와 Web pointerId가 달라서 전환이 무시되는 문제를 수정했습니다.
- 문서 검색 창을 닫은 직후 S Pen 버튼 지우개를 사용해도 검색 창이 다시 살짝 열리거나 화면을 미는 문제를 막았습니다.
- 중복으로 뜨던 검정 S Pen 지우개 HUD를 제거하고 파란 네이티브 칩만 남겼습니다.

## 두께와 HUD

- 확대 시 필기와 도형이 캔버스 해상도 제한 때문에 깨져 보이지 않도록, 고배율에서는 필기/도형을 SVG 벡터 오버레이로 렌더링합니다.
- PDF/이미지 배경 페이지의 캔버스 픽셀 예산은 유지해 고배율에서 메모리 사용량이 폭증하지 않도록 했습니다.
- 펜 획과 도형 윤곽선 두께를 렌더링 시점의 화면 픽셀 기준으로 다시 환산해, 같은 px 값이 확대율에 따라 다르게 보이는 문제를 줄였습니다.
- 설정에 **HUD 텍스트 투명도** 슬라이더를 추가했습니다.
- S Pen 지우개 알림, 네이티브 상태 표시, 스타일러스 정보 글자의 투명도에 같은 설정을 적용합니다.

## 검증

- 웹 회귀 테스트를 3.3.23 기준으로 갱신했습니다.
- S Pen 버튼 지우개 중간 전환, 검색 창 재등장 방지, 좌상단 HUD 위치와 자동 숨김, HUD 투명도 저장, 확대 후 화면 기준 두께, 고배율 벡터 필기 오버레이를 확인했습니다.
