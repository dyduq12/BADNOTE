# 3.3.23 변경 내역

## S Pen 지우개

- S Pen 버튼 지우개 알림을 화면 가운데에서 왼쪽 위로 옮겨 필기 영역을 덜 가리게 했습니다.
- 버튼을 놓거나 입력이 취소되면 지우개 알림이 자동으로 사라지도록 정리했습니다.
- 펜이 화면에 닿아 이미 획을 그리고 있는 중에도 S Pen 버튼을 누르면 즉시 해당 입력을 지우개 세션으로 바꿉니다.
- 문서 검색 창을 닫은 직후 S Pen 버튼 지우개를 사용해도 검색 창이 다시 살짝 열리며 화면을 밀지 않도록 막았습니다.

## 두께와 HUD

- 펜 획과 도형 윤곽선 두께를 렌더링 시점의 화면 픽셀 기준으로 다시 환산해, 같은 px 값이 확대율에 따라 다르게 보이는 문제를 줄였습니다.
- 설정에 **HUD 텍스트 투명도** 슬라이더를 추가했습니다.
- S Pen 지우개 알림, 네이티브 상태 표시, 스타일러스 정보 글자의 투명도에 같은 설정을 적용합니다.

## 검증

- 웹 회귀 테스트를 3.3.23 기준으로 갱신했습니다.
- S Pen 버튼 지우개 중간 전환, 검색 창 재등장 방지, 좌상단 HUD 위치와 자동 숨김, HUD 투명도 저장, 확대 후 화면 기준 두께를 확인했습니다.
