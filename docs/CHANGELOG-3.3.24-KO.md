# 3.3.24 변경 내역

## 버전

- 표시 버전을 `3.3.24`로 올렸습니다.
- Android 업데이트 설치가 가능하도록 `versionCode 356`으로 올렸습니다.

## 아이콘

- 배드노트 앱 아이콘을 새 불꽃 노트 아이콘으로 변경했습니다.
- 웹/PWA 아이콘과 Android launcher/round launcher 아이콘을 같은 원본으로 갱신했습니다.

## 확대 필기 품질

- 고배율 확대에서 필기와 도형이 깨져 보이지 않도록 SVG 벡터 오버레이 렌더링을 유지합니다.
- PDF/이미지 배경 페이지의 캔버스 픽셀 예산은 유지해 고배율에서 메모리 사용량이 폭증하지 않도록 했습니다.

## S Pen 지우개

- `ACTION_BUTTON_RELEASE`가 버튼 bit를 포함한 채 들어와도 S Pen 버튼을 놓은 것으로 처리합니다.
- 터치 중 S Pen 버튼을 눌렀을 때 Android pointerId와 Web pointerId가 달라 지우개 전환이 무시되던 문제를 수정한 상태를 유지합니다.
- 문서 검색 창을 닫은 직후 S Pen 버튼 지우개를 사용해도 검색 창이 다시 살짝 열리거나 화면을 밀지 않도록 막은 상태를 유지합니다.

## 검증

- 웹 회귀 테스트를 3.3.24 기준으로 갱신했습니다.
- 새 앱 아이콘, 고배율 벡터 필기 오버레이, S Pen 버튼 지우개 회귀 테스트, APK 서명/정렬/버전 검증을 확인했습니다.
