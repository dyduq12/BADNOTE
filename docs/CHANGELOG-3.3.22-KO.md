# 3.3.22 변경 내역

## 편집 화면

- 우측 필기 영역에 뜨던 세로 페이지 스크롤 레일을 제거했습니다.
- 페이지 번호 이동 입력은 페이지 창 상단으로 옮겼습니다.
- 페이지 창을 열면 메인 페이지가 사이드바 오른쪽 작업 영역 기준으로 다시 정렬됩니다.

## 페이지 창

- 페이지 썸네일을 한 줄에 2페이지씩 보이도록 그리드로 변경했습니다.
- 썸네일 카드, 페이지 번호, 객체 수, 페이지 메뉴를 분리해 잘못 눌리는 경우를 줄였습니다.
- 대용량 문서에서는 생략된 페이지 구간을 페이지 번호 입력으로 이동하도록 안내합니다.

## 렌더링

- 텍스트, 스티키 노트, 수식 객체의 글자 크기를 기준 페이지 폭으로 보정했습니다.
- 같은 pt 값이 화면 폭이나 사이드바 열림 상태에 따라 다르게 보이는 문제를 줄였습니다.

## 검증

- 웹 회귀 테스트를 새 페이지 창 동작 기준으로 갱신했습니다.
- 우측 스크롤 레일이 DOM에 남지 않는지, 2열 페이지 창과 페이지 번호 이동이 동작하는지 확인합니다.
